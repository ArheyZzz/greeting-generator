# 🎄 Greeting Generator (AI-Powered)

Современный генератор поздравлений, созданный для того, чтобы превратить сухие пожелания в душевные и персонализированные тексты. Проект интегрирован в экосистему [MakerPort](ваша_ссылка_на_портфолио).

## ✨ Особенности

* **AI-Core**: Генерация уникальных текстов с помощью нейросетей.
* **Стилизация**: Выбор настроения — от формального до «с юмором».
* **Dark Mode**: Интерфейс выполнен в премиальном темном стиле с элементами Glassmorphism.
* **Адаптивность**: Идеально работает на десктопах и смартфонах.

## 🤖 Роль ИИ в разработке
Этот проект был реализован в тесном сотрудничестве с **Gemini AI**. ИИ выступал не только в роли консультанта, но и:
* **Архитектор UI**: Мы вместе проектировали темную тему и Neo-Mint стилистику.
* **Code Reviewer**: Оптимизация CSS-структуры и логики компонентов.
* **Brainstorm Partner**: От идеи «сделать генератор» до финального релиза на GitHub Pages.

## 🛠 Технологический стек
* **React** (Vite)
* **CSS3** (Custom Properties, Glassmorphism, Flexbox/Grid)
* **GitHub Pages** (Deployment)
